# 儿童互动阅读软件产品愿景

## 一句话定位

一款面向儿童的互动阅读软件，把“阅读内容 + 游戏化任务 + 读后反馈 + 家长管理”整合成一个轻量产品。

## 不做什么

- 不复制现有阅读软件的品牌、课程、IP、图片、音频、绘本内容。
- 不直接复刻任何现有产品页面。
- 不把儿童内容做成纯刷屏娱乐产品。
- 不优先做复杂社交。

## 目标用户

### 儿童用户

- 年龄：3-12 岁。
- 需求：内容有趣、操作简单、能听、能点、能获得奖励。

### 家长用户

- 需求：知道孩子读了什么、读了多久、有没有理解、是否过度使用屏幕。

## 产品原则

1. 低门槛：孩子打开后 3 秒内知道点哪里。
2. 短任务：单次阅读任务控制在 5-15 分钟。
3. 强反馈：读完必须有星星、徽章、进度或鼓励反馈。
4. 护眼：默认加入休息提醒，不鼓励长时间连续使用。
5. 内容合规：使用原创、授权或公版内容。

## MVP 成功标准

- 能展示一个分龄书架。
- 能进入一本书的阅读页面。
- 能完成阅读任务。
- 能完成 2-3 道读后练习。
- 能展示阅读完成反馈。
- 能在家长中心看到阅读记录。
