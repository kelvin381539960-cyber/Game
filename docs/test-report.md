## 4. 本次修正记录

### 4.1 补充 TypeScript 类型依赖

检查 `package.json` 时发现，React + TypeScript 项目应补充 React 类型声明依赖。

已补充：

```json
{
  "devDependencies": {
    "@types/react": "latest",
    "@types/react-dom": "latest"
  }
}
```

同时将构建工具依赖调整到 `devDependencies`：

```json
{
  "devDependencies": {
    "@vitejs/plugin-react": "latest",
    "typescript": "latest",
    "vite": "latest"
  }
}
```
