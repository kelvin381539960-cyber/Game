# Game

一个儿童互动阅读软件原型，定位为“像游戏一样读书”的 Web 阅读产品。

> 说明：本项目借鉴儿童阅读产品的通用产品形态，例如分龄内容、互动阅读、每日计划、练习反馈、积分激励、家长控制等；不复制任何现有品牌的课程内容、IP、美术、文案或商业名称。

## 产品方向

目标用户：3-12 岁儿童及其家长。

核心目标：

- 让孩子愿意打开阅读内容。
- 通过互动和反馈降低阅读门槛。
- 通过每日任务和激励机制提高坚持率。
- 给家长提供进度、时长、护眼和内容管理能力。

## MVP 模块

1. 分龄书架：按年龄段推荐绘本/故事/知识内容。
2. 今日阅读计划：每天 15 分钟任务。
3. 互动阅读器：分页阅读、语音播放、互动问题、重点词解释。
4. 读后练习：选择题、跟读、理解题。
5. 激励系统：星星、徽章、连续阅读天数。
6. 家长中心：阅读报告、使用时长、护眼休息。
